# Frontend

React-based web application for parking predictions.

## Setup

```bash
npm install
npm run dev
```

## Environment Variables

Create `.env` file:
```env
VITE_API_URL=http://localhost:8001/api/v1
```

## Build

```bash
npm run build
```

## Features

- Interactive map with Leaflet
- User authentication
- Real-time predictions
- Responsive design
